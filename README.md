# Vary

Vary is a statically typed language that compiles to JVM bytecode with first-class mutation testing support.

- **Docs:** [varylang.com/docs](https://varylang.com/docs/)
- **Getting started:** [varylang.com/docs/getting-started](https://varylang.com/docs/getting-started/)
- **Install:** [Docker](https://varylang.com/docs/install-docker/) | [Linux](https://varylang.com/docs/install-linux/) | [Windows](https://varylang.com/docs/install-windows/) | [VS Code](https://varylang.com/docs/vscode/)
